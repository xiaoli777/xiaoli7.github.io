单反拍照技巧

Posted on 2020-12-29 In photography Views: Disqus:
本篇主要介绍，在具体场景下进行摄影，如何使用单反相机进行摄影。

Q1: 如何拍近距离照片？

AV档 + 最大光圈 + ISO自动 + 单点对焦

注：最近对焦距离 和 闪光灯开关

Q2: 如何抓拍运动物体？

TV档 / 调高快门速度 + AUTO自动感光度(ISO) + 自动对焦 + 高速连拍

Q3: 如何拍花卉？

大光圈镜头 / 长焦镜头 (虚化好)

广角俯拍大场景

远距离低角度拍 Q1

特写 + 人工露珠 Q1

逆光低角度仰拍脉络：点测光对准比较亮的花瓣

正上方俯拍：花蕊 + 花瓣

Q4: 如何拍创意光绘？

画面左右反向

手电筒 + 小皮筋 + 彩布 = 彩光手电筒

TV档(长快门时间) + 最低ISO + 手电筒对相机镜头

Q5: 汽车尾灯拉丝

选择车流较多，地势较高的天桥或街道

三脚架 + Tv档 + 5-20s快门 + ISO为100
