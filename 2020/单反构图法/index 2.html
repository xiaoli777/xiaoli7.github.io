单反构图法

Posted on 2020-12-29 In Photography Views: Disqus:
本篇介绍构图目的，以及在使用单反相机（SLR cemera）的过程中, 如何构图来突出主体，并表现出理想的效果。

构图目的

  • 突出被摄主体: 观者的视线能被吸引到被摄主体上

  • 画面简洁: 只保留必要元素，消除或减少分散观者注意力到元素

如何构图

  • 主体要突出
    • 调整虚实
    • 控制明暗
    • 改变焦距
  • 陪体有力衬托主体
    • 渲染气氛
    • 增加画面趣味性
    • 与主题形成内容呼应
    • 平衡画面，增加空间感
  • 背景衬托
    • 好的背景要简洁
    • 好的背景可以交代环境信息，并与主体形成呼应

构图方法

  • 三分构图法
    • 横向(风光照片)
      • 天际线在画面上下方1/3
    • 竖向(人像照片)
      • 主体人物在左右侧1/3
  • 九宫格构图法
    • 将需要表现的主体放在九宫格的4个交叉点上
  • 线条构图法
    • 曲线: 不呆板，延伸画面空间感
      • 小路 / 河流 / 女性身体曲线
    • 斜线: 力量与方向感，增强主体的气势和视觉冲击力
      • 运动体 / 人物身体线条
    • 汇聚线: 中心点
      • 视线 / 运动趋势
  • 形状构图法
    • 三角形构图: 稳定性
    • 弧线形构图: 柔美、生动
    • 框架构图
      • 风光摄影: 修饰主题 / 浓缩远景 / 汇聚视线
      • 人像摄影: 强化画面的空间感 / 丰富画面故事感
      • 屋檐 / 门框 / 桥洞 / 树枝
  • 对称构图
    • 将画面分割成对称的两部分: 平衡 / 稳定 / 相互呼应
    • 庄重而平稳的建筑物 / 正面人像 / 倒影的场合
    • 风光摄影: 水面的倒影
    • 人像摄影: 一左一右对称
  • 对比构图
    • 突出主体，让照片更加戏剧化、富有艺术感染力
    • 视觉反差强烈的色彩对比
      • 冷(蓝 / 绿): 使人后退
      • 暖(黄 / 红): 使人前进
    • 影调丰富的明暗对比
      • 将视觉中心汇聚到画面中的明亮处，并增加空间感
    • 给人想象空间的虚实对比
      • 以虚映实，突出清晰的主体，并延伸空间感，创造画面故事感
    • 夸张效果的大小对比
      • 丰富视觉表现力
    • 画面舒展的疏密对比
      • 厚实 / 密不透风: 沉重和压抑
      • 稀疏 / 松散: 轻松的自然美感
      • 疏而不散 / 密而不乱: 松弛有度的节奏感和韵律感
    • 动感强烈的动静对比
      • 容易引起观看者的强烈关注
      • 快门速度快(>1/2s): 动静结合
      • 快门速度慢(<1/20s): 人影飘忽
  • 留白让照片更有想象空间
    • 单一色调烘托画面意境，延伸画面空间
    • 天空 / 水面 / 浓雾 / 草原 / 人物视线
  • 善用前景营造唯美梦幻意境
    • 虚化前景，实现梦幻的画面意境
    • 树叶 / 花朵: 安排在画面的边缘
